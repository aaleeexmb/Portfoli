
# Gestión Avanzada de Usuarios y Seguridad del Dominio

Este proyecto se centra en la administración avanzada de un dominio **Active Directory** sobre **Windows Server**, simulando la estructura de una empresa real con distintos departamentos. Se ha creado una jerarquía de **Unidades Organizativas (OU)**, usuarios y grupos de seguridad, aplicando **políticas de contraseñas avanzadas (Fine-Grained Password Policies)** según el nivel de seguridad requerido por cada rol.

Además, se han implementado **Políticas de Grupo (GPO)** específicas para cada departamento, controlando el entorno de trabajo de los usuarios, y se ha realizado **delegación de administración**, auditoría avanzada y análisis de registros de seguridad. Este trabajo es especialmente útil en entornos empresariales, ya que permite garantizar el control de accesos, la trazabilidad de acciones y una gestión eficiente y segura de los usuarios del dominio.
