
 Bash Scripts Workshop

Este proyecto recopila distintos **scripts desarrollados en Bash** orientados a la automatización de tareas habituales en sistemas Linux. Los scripts realizan funciones como gestión básica del sistema, manipulación de archivos, control de procesos y ejecución de comandos de forma automatizada, con el objetivo de reducir tareas repetitivas y mejorar la eficiencia del administrador del sistema.

Las tecnologías utilizadas en este proyecto son **Bash**, **Linux** y utilidades estándar del sistema como `grep`, `awk`, `sed`, `chmod` o `cron`. Estos scripts son aplicables en entornos reales de administración de sistemas, ya que permiten automatizar mantenimientos, facilitar la gestión de servidores y servir como base para tareas de **DevOps** o **SysAdmin**.
